## Budgeting App

### Built With

* Django
* SQLite
* HTML/CSS
* Bootstrap

### Setup

1. Create a superuser for yourself.  This is necessary for sending an automated warning email
2. Start the server 
   ```.sh
   python manage.py runserver 
   ```
